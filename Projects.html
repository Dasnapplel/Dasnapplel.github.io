Swordfight Shenanigans (May 2023 - May 2024)

This project is a multiplayer online battle arena game where players control customizable characters and engage in fast-paced sword fights. It is a Roblox game that has around 100 different swords that players can battle with, and 6 different maps to choose from. I created it as my highschool capstone project, to showcase my skills in game development. Due to this, every aspect of the game, including the code, soundtrack, models, and UI, were created by me alone. Since this was my first big personal project, I learned a lot about game development, coding, and how to work with a large codebase. The game is still unfinished, but I plan to add more features and polish it further once I improve my skills as a computer scientist and videogame developer.

Python Quest Game (November 2024 - December 2024)

This was my final project for CIS 162, intro to computer science, at GVSU. We were to create a quest game using everything that we learned on python so far, that included items, npcs, different rooms, and a way to beat the game. My project was set in the backrooms, a scary and dangerous place that must be escaped from. What’s unique about my project is that all of the rooms randomly connect to each other, so you never know where you’ll go. Additionally, there are three different endings depending on how you play the game. I recieved a score of 100% along with extra credit for implementing colored text. I’m proud of what I was able to create!

Pokemon Battle Assistant (July 2025)

Over the duration of my four week study abroad program in Switzerland, we had to create a project that included machine learning models and image recognition. My group’s project was to create a pokemon GO battle assistant. The first part of the project was to scrape data from thousands of simulated pokemon battles, and create a data model to accurately predict who would win a new battle when given two pokemon. The second part of the project was to create an image recognition model that could identify any pokemon when given an image of it. Our group was split evenly between American and Swiss students, along with our work. I worked on the pokemon battle predictor and UI of the program. Together, we won the prize for best presentation in our class!
