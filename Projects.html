Gaming

Swordfight Shenanigans (May 2023 - May 2024)

This project is a multiplayer online battle arena game where players control customizable characters and engage in fast-paced sword fights. It is a Roblox game that has around 100 different swords that players can battle with, and 6 different maps to choose from. I created it as my highschool capstone project, to showcase my skills in game development. Due to this, every aspect of the game, including the code, soundtrack, models, and UI, were created by me alone. Since this was my first big personal project, I learned a lot about game development, coding, and how to work with a large codebase. The game is still unfinished, but I plan to add more features and polish it further once I improve my skills as a computer scientist and videogame developer.

Overdrive (August 2024 - November 2024)

This project is a car crash derby battle royale game where players compete in chaotic vehicular mayhem. This Roblox game was made for a game jam, where competitors had to create a game in just 2 weeks. My brother and I collaborated on this project, with him handling the art and modeling while I focused on the programming and game mechanics. Although we didn't win, and the game is still incomplete, we are both proud of what we accomplished in such a short time frame.

Glitch Maze (Summer 2022 and Fall 2025)

I really like creating games on Roblox Studio. One of the projects that I made randomly at some point was Glitch Maze. This game resembles PacMan in a lot of ways, such as the maze, pellets, and enemies that roam around the maze. However, my game is in first person, features music made by me, and focuses more on the fear of the maze. Additionally, I revisited the game during my Freshmen year in college, and polished it using new skills that I learned in my Computer Science classes.

Academic

Python Quest Game (November 2024 - December 2024)

This was my final project for CIS 162, intro to computer science, at GVSU. We were to create a quest game using everything that we learned on python so far, that included items, npcs, different rooms, and a way to beat the game. My project was set in the backrooms, a scary and dangerous place that must be escaped from. What’s unique about my project is that all of the rooms randomly connect to each other, so you never know where you’ll go. Additionally, there are three different endings depending on how you play the game. I’m proud of what I was able to create!
